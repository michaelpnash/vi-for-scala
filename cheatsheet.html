Key bindings for my Vi setup for editing Scala, available here.

Vi
Buffers
,b
 Next Buffer
,B
 Previous Buffer
:buffers
List all current buffers
Windows
,oJ
Split window, open test in other window
:split
Split window
,sw
Size current window to fit lines in file (for small files)
C-w=Make all windows the same size
C-w-
Make current window smaller by one line (can prefix with number)
C-w+
Make current window larger by one line (can prefix with number)
:new XXX
Open file xxx in new window. Prefix with number to set size of window
QuickFix
\ff
Jump to first compile error
\fn
Jump to next compile error
Tags
:tag XXX
Jump to definition of symbol XXX
C-]
Jump to definition of tag under cursor
C-t
Jump back to point before jumping to definition
NERDTree
freespec C-\
ScalaTest Freespec
C-n
Toggle NERDTree window

Control-P

Key Command
,cd Change working directory to directory of current file
,g Prompt to grep in subdirectories for a pattern (:Grep)
<f3>
 Pass current file through Scalariform, formatting it (:Autoformat)

Tmux

C-a C-up, C-a C-left, C-a C-right resize by 1 row/column
C-a C-down
resize smaller by 1 row
tmux new-session -s work Start a new session
tmux attach -t work Attach to an existing session
C-a $ rename the current session
C-a ( previous session
C-a ) next session
C-a L last (previously used) session
C-a s choose a session from a list
C-a c create a new window
C-a 1 ... switch to window 1, ..., 9, 0
C-a p previous window
C-a n next window
C-a l last (previously used) window
C-a w choose window from a list
C-a left go to the next pane on the left
C-a right (or one of these other directions)
C-a o go to the next pane (cycle through all of them)
C-a ; go to the last (previously used) pane
C-a o Switch to other pane
C-a M-down Make current pane 5 lines smaller
C-a " Split vertically
C-a % Split horizontally
